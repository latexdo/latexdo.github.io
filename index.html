      Install the local LatexDo editor with one command.

      The CLI keeps a cached copy of the LatexDo source, installs npm dependencies when they change, starts the local web editor, and opens the localhost URL in your browser.

      Direct installers

      Downloads are versioned for every LatexDo release.

      Installers are published at latexdo.org/downloads with a manifest and checksums for the application updater. GitHub Releases store the versioned installer assets.

      Questions

      Answers for writers, reviewers, and maintainers.

      Can people download LatexDo without a GitHub account?

      Yes. The public downloads page hosts installers, checksums, and the update manifest directly on the LatexDo website.

      Is the web editor the same ecosystem as the desktop app?

      Yes. The hosted editor is the fast entry point, while the desktop app handles local compilation, PDF export, terminal access, and project files.

      Who is LatexDo for?

      Researchers, students, technical writers, reviewers, and anyone who wants a focused LaTeX workflow for source, PDFs, citations, and rebuttals.
