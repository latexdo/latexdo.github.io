About LatexDo

A focused LaTeX workspace for serious writing.

LatexDo is built for people who write technical documents and need the source, PDF, diagnostics, references, reviews, and release-ready output to stay connected. It is not a generic note app or a thin file viewer; it is a writing environment shaped around LaTeX projects.

What it is

LatexDo keeps the paper workflow in one place.

The web editor gives a fast way to start writing. The desktop app adds local project folders, terminal access, SyncTeX-aware PDF preview, and compilation through your own TeX installation.

Write with source control

Edit real .tex and .bib files in a focused project workspace while keeping the file tree, diagnostics, and output close by.

Compile locally

The desktop app works with your local LaTeX toolchain, so papers, figures, bibliographies, and generated PDFs stay on your machine.

Handle review work

LatexDo includes tools for comments, rebuttals, citation checks, notation cleanup, and submission-focused document checks.

Why it exists

LaTeX writing has more state than a text editor shows.

A paper is source files, references, generated PDFs, compile logs, review notes, submission rules, rebuttal drafts, and small correctness checks. LatexDo brings those parts into a single interface so authors can move from draft to final submission without constantly switching tools.

The goal is practical: make technical writing feel direct, inspectable, and local when it matters, while still offering a web editor for quick access.
